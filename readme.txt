== Description ==
This plugin allows to receive WPPizza online-shop orders directly into WinOrder POS software.
- [WPPizza](https://wordpress.org/plugins/wppizza) is a great online shop system Wordpress-Plugin for delivery services
- [WinOrder](https://www.winorder.com) is a point-of-sale desktop solution for delivery services


Ordered list:

1. Attach WPPizza online shop order as JSON file (for shop email-templates only). Process online orders directly in WinOrder
2. Synchronize articles and settings with WinOrder
3. Send order status to WPPizza online shop


== Installation ==
1. Upload "winorder-wppizza-connector.php" to the "/wp-content/plugins/" directory
2. Make sure WPPizza plugin is installed and activated in Wordpress
3. Make sure XML-RPC is enabled in Wordpress
4. Activate the plugin through the "Plugins" menu in WordPress
5. Enable the "shop" email-template in WPPizza settings
